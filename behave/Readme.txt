To run the Software Regression test follow these steps:

Step-1 Go to the directory where the features folder is located

Step-2 Open the terminal here

Step-3 run this command: behave <Name_of_the_featureFile_you_want_to_run>.feature

Step-4 This starts running the feature file

Step-5 To run Step-23 from the software regression test, run the Step24_SWRegressionTest.py file,
       then it gives you a trace of all the steps executed to pass the Step-24
